# PawliciousDonationDashboard
A self development Power BI project integrating by MySQL and Excel for donation data analysis and visualization..

# Project Background
After researching internship and entry-level positions in data analytics, I observed that Power BI is one of the most frequently requested skills in the industry. As it was not covered in my university curriculum, I identified a gap in my skill set and took the initiative to learn it independently.

At the same time, I recognized an opportunity to apply my learning to a real-world dataset. Pawlicious, a student-led charity initiative at Universiti Utara Malaysia (UUM), managed donation records primarily through Microsoft Excel. The initiative raised funds to support the rescue and treatment of injured stray dogs on campus.

Although Pawlicious is no longer operating following my graduation, its historical donation records provided a valuable case study. The data had never been presented through a centralized, user-friendly dashboard, making it an ideal opportunity to practice Power BI while transforming raw donation records into meaningful and accessible business insights.

# Objective
1. Study, explore and strengthen my data processing and data visualizing skills.
2. Cleaning and preparing donation data.
3. Building an interactive Power BI dashboard.
4. Identify donation patterns and key performance indicators.

# Dataset
1. Historical donation record.

# Methodology
1. Microsoft Excel (data storing and preprocessing)
2. My SQL (data extraction, transformation and data cleaning)
3. Power BI (data visualization and dashboard development)

# Dashboard KPIs
- Total Donations
- Total Donors
- Average Donation Amount
- Donation Expenditure Trend
- Number of Dogs Rescued
- Treatment Categories
